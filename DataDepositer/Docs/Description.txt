План по Data Depositor

Основная цель программы: 
	Создание распределенного хранилища на компьютерах в локальной сети и Интернете.

Описания и принцип работы:
	Пользователь использует логин и пароль ( 2 ключа) для шифрования файла на своем компьютере.
	Затем файл отправляется на машины с установленным хранилищем, при этом файл разбивается 
	на несколько частей и кажадя часть отправляется на несколько компьютеров (по умолчанию 3 на 3)
